# Discord Fluxer Bridge

Complete re-write of an project I started earlier this year to attempt to replicate messages from one platform on the other to help communities make a migration to Fluxer without having members have a feeling of FOMO.

Rewriting because of "edge-cases" where I had not planned the project out properly from the beginning and ended up forgetting things such as bot messages, attachments, etc. The original codebase became spaghetti pretty quickly so learning from my mistakes and starting again.
